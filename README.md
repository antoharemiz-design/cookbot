
# 🍽️ CookBot — Ваш Шеф

Персональный AI-шеф-помощник в Telegram. Придумывает рецепты из ваших продуктов, составляет меню на неделю, подбирает вино и считает КБЖУ.

## 🔗 Ссылки

- **Бот в Telegram:** [@cookmaster_shef_bot](https://t.me/cookmaster_shef_bot)
- **Канал автора:** [@architectkulees](https://t.me/architectkulees)

## ✨ Возможности

### Основные
- 🍳 **Генерация рецептов** из продуктов, которые вводит пользователь (текстом или голосом)
- 🥗 **Учёт диеты, аллергий и нелюбимых продуктов** — один раз настроил, бот помнит
- 📅 **Планировщик меню** на день или неделю
- 🍷 **Подбор вина** к ужину
- 📊 **Расчёт КБЖУ** для каждого блюда
- 🧊 **Виртуальный холодильник** — бот готовит из того, что реально есть
- ⏱ **Быстрый ужин** — рецепт до 20 минут

### Геймификация
- 🗡 **Ежедневные квесты** (приготовь веганское блюдо, используй 3 ингредиента и т.д.)
- ⭐ **Очки и уровни** (Новичок → Умелец → Шеф → Мастер-шеф)
- 🏆 **Достижения** за прогресс
- 📖 **Дневник приготовленных блюд**

### Умные функции
- 📊 **Персональная аналитика** — бот запоминает любимые кухни и ингредиенты на основе оценок
- 📚 **Тематические коллекции** (Новогодний стол, Веганская неделя, Азиатская кухня и др.)
- 🔔 **Блюдо дня** — ежедневная рассылка рецепта
- 🌐 **Инлайн-режим** — работает в любом чате через `@cookmaster_shef_bot`
- 🔒 **Обязательная подписка** на канал автора

## 🛠 Технический стек

| Компонент | Технология |
|-----------|------------|
| Язык | Python 3.12+ |
| Фреймворк | Aiogram 3.x |
| AI-модель | Groq (llama-3.1-8b-instant) |
| База данных | SQLite (MVP) |
| Планировщик | APScheduler |
| Хостинг | Render (Web Service) |
| Health-check | cron-job.org |
| Middleware | Кастомная проверка подписки на канал |

## ⚙️ Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/antoharemiz-design/cookbot.git
cd cookbot
2. Установка зависимостей
bash
python3 -m venv venv
source venv/bin/activate  # для Linux/Mac
# venv\Scripts\activate   # для Windows
pip install -r requirements.txt
3. Настройка переменных окружения
Создайте файл .env в корне проекта:

text
TELEGRAM_BOT_TOKEN=ваш_токен_бота

4. Запуск
bash
python main.py
